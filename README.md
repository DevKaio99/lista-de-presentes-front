# 🎁 Lista de Presentes – Frontend

Aplicação web desenvolvida para gerenciar uma **lista de presentes de chá de casa nova**.

O objetivo do projeto é permitir que convidados visualizem os presentes disponíveis e contribuam escolhendo um item da lista e realizando o pagamento via PIX.

Este projeto foi desenvolvido como **projeto pessoal full stack**, com o objetivo de praticar desenvolvimento web, integração entre frontend e backend e versionamento com Git.

---

## 📌 Visão Geral do Projeto

A aplicação exibe uma lista de presentes disponíveis para um evento de chá de casa nova.

Os usuários podem:

- Visualizar os presentes disponíveis
- Escolher um presente para contribuir
- Realizar o pagamento via chave PIX exibida na aplicação

O frontend consome uma API responsável por gerenciar os dados da lista de presentes.

⚠️ Observação: O backend utilizado neste projeto não está atualmente hospedado online.

---

## 🚀 Funcionalidades

- Exibição da lista de presentes
- Seleção de presente pelo usuário
- Exibição de chave PIX para pagamento
- Integração com API REST
- Interface simples e intuitiva
